Algunos lenguajes de la web

HTML + CSS + Javascripts

  • HTML5

    HTML5 (HyperText Markup Language, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML.

    Usa la extensión.html

  • CSS

    Hoja de estilo en cascada o CSS(siglas en inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.

    Usa la extensión.css

    • JavaScript

    JavaScript (abreviado comúnmente JS) es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos,3 basado en prototipos, imperativo, débilmente tipado y dinámico.

    Usa la extensión.js
